GET /users
Params
id (optional) - number (example /users?id=5)
name (optional) - string (example /users?name="John")
Examples https://somesite.ru/users - return JSON with array of all users https://somesite.ru/users?id=5 - return JSON with user object https://somesite.ru/users?name="John" - return JSON with array of all users with name "John"
POST /users
Params
name (required) - string (example /users?name="John")
lastName (required) - string (example /users?lastName="Smit")
password (required) - string (example /users?password="123")
Examples https://somesite.ru/users?name="John"&lastName="Smit"&password="123" - return JSON with result of registration (true or false)
DELETE /users
Params
id (required) - number (example /users?id=1)
password (required) - string (example /users?password="123")
Examples https://somesite.ru/users?id=1&password="123" - return JSON with result of remove user (true or false)
